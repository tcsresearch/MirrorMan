# MirrorMan

> [!NOTE]
> A configurable utility to colorize reposync on Fedora.
<hr>

> [!WARNING]
> This code is experiemental and has NOT been fully implemented or tested yet.
<hr>

> [!IMPORTANT]
> Initial testing has been completed.  Not ready for production use and still lacks most features.
<hr>
Usage <br>
cecho <i>color</i> "text"
where color is one of the following: 

                black  | bk 
                red    |  r 
                green  |  g 
                yellow |  y 
                blue   |  b
                purple |  p 
                cyan   |  c 
                gray   | gr 
<p></p>

    Examples:
    cecho yellow "This text is yellow."
    cecho blue "This text is blue."
    cecho y "Yellow Text"
    cecho b "Blue Text"
